# IOT / Industrial Internet Framework [![Build Status](https://magnum.travis-ci.com/Ooppa/iot-industrial-internet.svg?token=UV8u6MHpWA2xq8xkw4ss&branch=dev)](https://magnum.travis-ci.com/Ooppa/iot-industrial-internet) [![Build Status](https://magnum.travis-ci.com/Ooppa/iot-industrial-internet.svg?token=UV8u6MHpWA2xq8xkw4ss&branch=master)](https://magnum.travis-ci.com/Ooppa/iot-industrial-internet)

The purpose of this software development project is to generate ideas and to develop an open source framework for reuse in Industrial Internet and Internet of Things related projects. The framework should offer the basic services needed for these types of projects. Examples of these services are gathering, analyzing, reporting and visualizing data.
